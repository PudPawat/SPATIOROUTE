SpatioRoute

Dynamic Prompt Routing for Zero-Shot Spatial Reasoning
Tailor each egocentric question to a specialized prompt template — with rule-based routing (SpatioRoute-R) or a lightweight text LLM with few-shots (SpatioRoute-L) — then answer with any vision-language model.

Paper (arXiv) Code Quick Start

Pawat Chunhachatrachai · Gueter Josmy Faure · Hung-Ting Su · Winston H. Hsu

Abstract

Spatial question answering over egocentric video requires vision-language models to reason about 3D layout, affordances, and viewer-relative directions — especially in the zero-shot setting. SpatioRoute routes each question to a semantically tailored prompt without fine-tuning or 3D input. On SQA3D, it improves overall accuracy by up to 5% over fixed prompts across multiple VLM families, and outperforms uniform chain-of-thought prompting on Qwen models.

No training
Off-the-shelf VLMs + optional small text router.
Video-only
Uniformly sampled frames; no point-cloud pipeline.
Question-aware
What / Is / Can / … → specialized templates.

SpatioRoute-R vs SpatioRoute-L

Figure 2 — Two complementary routing modes sharing the same typology table.
SpatioRoute-R

Deterministic map from SQA3D question type to YAML template. Fills Situation: and Question: lines. Fast and fully reproducible.

SpatioRoute-L

Same route + semantics flags, then a small text LLM (default Qwen2.5-0.5B) rewrites the prompt using few-shot YAML-layout examples. No video at routing time.

Quick Start

# Clone & install
git clone git@github.com:PudPawat/SPATIOROUTE-Dynamic-Prompt-Routing-for-Zero-Shot-Spatial-Reasoning.git
cd SPATIOROUTE-Dynamic-Prompt-Routing-for-Zero-Shot-Spatial-Reasoning
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt

# Prepare self-contained release (copies VLM eval deps from parent repo once)
./scripts/prepare_github_release.sh /path/to/Qwen_playground

# Smoke test (50 samples)
./scripts/run_smoke_test.sh

# Full SpatioRoute-R → VLM eval
python -m experiments.spatioroute.preprompts.generate_rule \
  --split test --output results/preprompts_r_test.json
python -m experiments.spatioroute.eval.routed \
  --preprompt-json results/preprompts_r_test.json \
  --model-name Qwen/Qwen2-VL-2B-Instruct --load-in-4bit \
  --output results/spatioroute_r_test.json

See the repository README for full instructions.
